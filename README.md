📊 COVID-19 Dashboard
📌 Overview

A COVID-19 dashboard is an interactive data visualization tool that tracks and displays real-time information about the spread of the virus, including confirmed cases, recoveries, deaths, and vaccination progress. It helps users understand trends over time and across regions for better awareness and decision-making.

👉 Dashboards typically combine data from sources like World Health Organization and Ministry of Health and Family Welfare to present reliable insights.

💡 Key Features
Total confirmed, active, recovered, and death cases
State-wise / country-wise analysis
Daily and weekly trend visualization
Vaccination progress tracking
Interactive charts, maps, and filters

👉 Many dashboards also include heat maps and trendlines to help users quickly understand how the virus spreads across regions

🎯 Purpose
Monitor pandemic spread in real-time
Support government and healthcare decisions
Provide public awareness and transparency
Analyze patterns and future trends
